# Dynamically HTML

> Dynamically HTML in HTML

## Install

```
$ npm install dynamically-html
```

## Usage

[example](test)

```
var dynamicallyHtml = require('dynamically-html');

dynamicallyHtml('<div>hello world</div>');
```
